# Forge 🔥

> **Forge your strength.** — a gamified strength-training companion for gym beginners.

Learn the core lifts, log your workouts, earn Bronze → Platinum medals as your lifts grow,
keep a training streak, and ask an AI coach grounded in real exercise knowledge and your own logs.

**Stack:** Next.js (App Router) · TypeScript · Supabase (Postgres + Auth) · Tailwind CSS v4 ·
shadcn/ui · Framer Motion · Recharts · OpenAI (Vercel AI SDK).

> 🚧 Work in progress — built milestone by milestone. A full case-study README with live demo,
> screenshots, and setup instructions lands at the end of the build.

## Local development

```bash
pnpm install
cp .env.example .env.local   # fill in Supabase + OpenAI keys
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
